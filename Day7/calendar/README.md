## 📍 배운점
### 1. `table` 태그
  - 사용하는 이유
    표를 만들기 위해서 
  - 사용하는 방법
    |태그|설명|
    |------|---|
    |`table`|표 삽입|
    |`tr`|표에 행 삽입|
    |`th`|표의 제목 셀 생성|
    |`td`|표의 일반 셀 생성|
    
    ```html
    <table>
      <tr>
        <th></th>
        <th></th>
        <th></th>
      </tr>
      <tr>
        <td></td>
        <td></td>
        <td></td>
      </tr>
    </table>
    ```
